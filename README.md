# Quentin Denoual — Site prototype

Structure minimale pour site vitrine (TCC & TERV) — prêt à être zippé et poussé sur GitHub Pages.

Pour tester :
1. Ouvrir `index.html` dans le navigateur.
2. Page ressources : utilise PDF.js pour générer vignette client-side (prototype).
3. Maquette image : ouvrir `assets/mockup/mockup.html` et cliquer sur "Télécharger en PNG".

Notes :
- En production, mettre en place un backend pour stocker PDF, générer vignettes serveurside et gérer RDV.
- Intégrer un système de rendez-vous (Calendly / Doctolib API / ou solution interne).
